🤖 AI-Powered Smart Investment Assistant with Sentiment \& Risk Prediction



📌 Project Overview



The AI-Powered Smart Investment Assistant is a web-based application that helps users make informed investment decisions using Artificial Intelligence and Machine Learning. It combines real-time stock market data, deep learning-based price prediction, news sentiment analysis, and investment recommendations in an interactive dashboard.



\---



🚀 Features



\- 📈 Real-time Stock Price Tracking

\- 🤖 AI-Based Stock Price Prediction using LSTM

\- 😊 News Sentiment Analysis

\- 💡 Buy / Hold / Sell Recommendation

\- 📊 Interactive Stock Charts

\- 💱 USD to INR Currency Converter

\- 🔍 Search Stocks Using Company Name or Stock Symbol

\- 🌐 User-Friendly Web Interface

🔗 API Integration



The application integrates multiple APIs to provide accurate and real-time financial information.



\- Yahoo Finance (yFinance) – Retrieves stock prices, historical data, and company information.

\- NewsAPI – Fetches the latest financial news articles.

\- TextBlob – Performs sentiment analysis on news articles.

\- Exchange Rate API – Converts USD to INR using live exchange rates.

🧠 LSTM Model



The application uses a Long Short-Term Memory (LSTM) deep learning model to forecast future stock prices using historical market data. The trained model ("lstm\_model.h5") generates stock price predictions that support investment recommendations.
