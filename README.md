 🤖 AI-Powered Smart Investment Assistant with Sentiment & Risk Prediction

## 📌 Project Overview

An AI-powered stock analysis and investment assistant built using Python, Flask, LSTM, TensorFlow, Yahoo Finance API, News API, and sentiment analysis. The application provides real-time stock prices, AI-based price forecasting, market sentiment analysis, investment recommendations, and currency conversion through an interactive web interface.

---

## 🚀 Features

- 📈 Real-time Stock Price Tracking
- 🤖 AI-Based Stock Price Prediction (LSTM)
- 😊 News Sentiment Analysis
- 💡 Buy / Hold / Sell Recommendation
- 📊 Interactive Stock Charts
- 💱 USD to INR Currency Converter
- 🔍 Search Stocks Using Company Name or Ticker Symbol
- 🌐 Web-Based User Interface
